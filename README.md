## 开发方式简述：

- 前端
  1. vue整合element-ui，axios
  2. store状态管理储存jwt
  3. axios全局拦截
  4. 博客编辑/发布接入mavon-editor
  5. 文章详情markdown渲染

  
- 后台

  1. SpringBoot整合MybatisPlus
  3. 整合shiro-redis，会话共享
  4. shiro整合jwt，身份校验
  5. 实体校验
  6. 跨域
